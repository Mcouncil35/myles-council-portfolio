# Myles Council Portfolio

Personal portfolio website showcasing my skills, experience, and AI expertise.

## Features

- Modern UI with animations and neural network visualization
- Interactive AI assistant (Helix) for visitor inquiries
- Responsive design for all devices
- Skills section with progress bars
- Experience timeline with detailed highlights

## Tech Stack

- React 19
- Vite
- CSS3 with custom animations

## Live Site

https://mcouncil35.github.io/myles-council-portfolio/

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
